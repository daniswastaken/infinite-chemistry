# Contributing to Infinite Chemistry

First off, thank you for considering contributing to Infinite Chemistry! This project is built to help students learn chemistry stoichiometry and nomenclature in a fun, interactive way.

## How to Contribute

### 1. Found a Bug?

If you find a bug (like a wrong chemical formula or a UI glitch), please open an Issue on GitHub with:

- A clear description of the bug.
- Steps to reproduce it.
- Expected behavior vs. actual behavior.

### 2. Feature Requests

Have an idea for a new feature? Maybe a specific chemical reaction or a new UI effect? Open an Issue to discuss it!

### 3. Pull Requests

1. **Fork** the repository.
2. Create a **feature branch** (`git checkout -b feature/AmazingFeature`).
3. **Commit** your changes (`git commit -m 'Add some AmazingFeature'`).
4. **Push** to the branch (`git push origin feature/AmazingFeature`).
5. Open a **Pull Request**.

## Development Guidelines

- **Chemistry Accuracy:** Ensure all new bonding logic follows standard chemistry rules.
- **Localization:** Current default language is **Indonesian**. If adding new terms, ensure they match Indonesian nomenclature conventions.
- **Styling:** We use **Tailwind CSS**. Try to keep the UI clean and minimalist.
- **Vue 3:** Use the **Composition API** and follow Vue best practices.

## Code of Conduct

To ensure a positive learning environment, all contributors are expected to follow our [Code of Conduct](CODE_OF_CONDUCT.md).
