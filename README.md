# Infinite Chemistry

Infinite Chemistry is an open source implementation of the popular game Infinite Craft but with chemistry nomenclature teaching for study as the main focus. 

# How This Game Works
The mechanics still the same, using drag and drop. The user will be provided with a periodic table of elements available.

The user then can drag and drop them together. For example:
The user draggin Nitrogen (N) and Oksigen (O) to form Nitrogen Oksida (NO).

The elements name are in Indonesian.
